# Premium Restaurant Menu Landing Screen

A modern, fast, mobile-first QR menu landing screen built with React, Tailwind CSS, and Framer Motion.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

**Note:** This UI is designed specifically for mobile devices. For the best experience, use your browser's Developer Tools to toggle device emulation (Mobile View).

## ✨ Features
- **Modern UI/UX**: Clean, high-end design similar to top-tier food delivery apps.
- **Multilingual Support**: Supports English and Arabic (RTL) with a toggle in the header.
- **Popular Items & Categories**: Quick browsing of menu sections.
- **Interactive Cart**: Add/remove items with a floating cart button and a sliding drawer.
- **Premium Animations**: Smooth transitions and touch-feedback using Framer Motion.

## 🧪 Testing
Run the UI verification tests with Playwright:
```bash
npx playwright test
```
